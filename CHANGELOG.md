# CHANGELOG

所有重要的更改都会记录在这个文件中。

版本号格式：`0.0.x`
- 每次功能更新递增 x
- main 分支包含稳定版本
- dev 分支用于开发

---

## [0.0.3] - 2026-03-31

### 修复
- 修复翻译结果显示 `Parse("No translation result")` 错误
- 修复标题栏引擎下拉框无法点击选择的问题
- 修复引擎选择在标题栏和设置窗口之间不同步的问题
- 修复外观设置修改后翻译窗口不生效的问题

### 功能增强
- **Google 翻译双 URL 支持**：镜像源 URL + 官方 API URL + API Key，自动降级
- **设置自动保存**：2 秒防抖自动保存，关键设置（引擎/语言）立即保存
- **跨窗口设置同步**：修改引擎或语言后，翻译窗口实时响应
- **错误提示优化**：引擎错误信息改为中文友好提示，凭证缺失时提前校验

### 技术改进
- 后端配置向后兼容：旧版 `google_url` 自动迁移到新字段
- 前端事件监听：`theme-changed` 事件实现跨窗口状态同步
- Radix UI 下拉框修复：`onClick` 改为 `onSelect` + `e.preventDefault()`

## [0.0.2] - 2026-03-30

### 设置界面改进
- 引擎设置改为可折叠手风琴按钮，图标移至右侧
- 输入框高度缩小，背景与设置页一致，白色底边指示
- 聚焦时主题色从中间向两边扩散填充底边
- 下拉框样式：背景与页面一致，白色底边，下拉框 #1C1D21

### 功能增强
- 百度翻译增加 URL 输入框
- 谷歌翻译默认 URL: https://translate.googleapis.com/translate_a/single
- 百度/谷歌 hover 弹窗提示
- 通用设置增加重置按钮（重置主题、引擎、语言、API 密钥）

## [0.0.1] - 2026-03-30

### 修复
- app外观功能修复

### 修复详情
1. **跨窗口主题同步**：设置窗口修改主题后，翻译窗口立即更新，无需重启应用
2. **窗口透明度**：翻译窗口现在可以看到下方其他应用的窗口
3. **标题栏透明度映射**：标题栏透明度范围为文本显示区域的一半（0%-50%）
4. **背景透明度**：使用RGBA背景色替代opacity属性，确保文字清晰可读

## [0.0.0] - 2026-03-29

### 添加
- 初始化 Tauri + React 翻译应用项目
- 实现基础翻译功能（百度、Google、Ollama、SiliconFlow）
- 添加 shadcn/ui 组件库
- 实现剪贴板监听功能
- 添加主题切换（亮色/暗色）
- 实现设置界面（引擎配置、快捷键、主题等）
- 添加 LaTeX 数学公式渲染支持

### 技术栈
- **前端**: React 19 + TypeScript + Tailwind CSS
- **后端**: Rust (Tauri 2)
- **UI 组件**: shadcn/ui (Radix UI)
- **构建工具**: Vite
