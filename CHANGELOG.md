# CHANGELOG

所有重要的更改都会记录在这个文件中。

版本号格式：`0.0.x`
- 每次功能更新递增 x
- main 分支包含稳定版本
- dev 分支用于开发

---

## [0.0.7] - 2026-05-25

### 新功能
- 新增划词翻译基础能力：支持独立 `selection-popup` 悬浮窗口、划词设置页、多入口构建和后端划词触发模块。

### 修复
- 修复开启划词翻译后没有弹窗的问题：设置保存后立即同步后端运行态，触发时如果弹窗窗口不存在会自动补建。
- 修复弹窗首次创建时可能错过划词事件的问题：后端缓存最近一次划词内容，前端启动后主动拉取并渲染。
- 修复自动划词鼠标监听只处理一次消息后退出的问题。

---

## [0.0.6] - 2026-05-02

### 优化
- **设置界面弹窗（Tooltip）重构**：所有引擎设置弹窗从 `position: absolute` 改为 `position: fixed`，避免被滚动容器截断
- **弹窗自动向上弹出**：检测到下方空间不足时自动向上弹出，右侧溢出时自动左移
- **长文本拆行显示**：百度 APP ID/密钥、谷歌镜像源/官方 URL、大模型API 地址/模型等长文本弹窗拆成两行
- **精简弹窗**：仅举例输入框（URL、模型等）保留弹窗提示，API Key、VLM 模型等移除弹窗
- **图片翻译标签**：移除「仅LLM引擎」标注，改为通用「图片翻译」

### 新功能
- **大模型API URL 可配置**：新增 `llmapi_url` 设置字段，支持任何 OpenAI 兼容 API 提供商（DeepSeek、Moonshot、智谱、Groq、Together、LM Studio、vLLM 等）

### 修复
- 修复设置窗口弹窗被 `overflow-y-auto` 截断的问题（`settings.html` 添加 `overflow: visible`）
- 修复设置窗口根容器 `overflow-hidden` 导致弹窗无法显示的问题

---

## [0.0.5] - 2026-05-01

### 新功能
- **最顶层显示**：通用设置新增开关（默认开启），控制翻译窗口是否始终在其他窗口上方，Tooltip 提示，切换即时生效
- **自动显示**：通用设置新增开关（默认开启），翻译完成后自动弹出窗口并显示翻译内容（最小化→恢复聚焦，未最小化→置顶）
- **图片翻译（VLM）**：通用设置新增开关（仅 LLM 引擎），大模型API/Ollama 支持 VLM 模型图片翻译
- **设置文件路径迁移**：从 `%APPDATA%/translate_app/settings.json` 迁移至 `~/translate_app.json`，自动检测并迁移

### 优化
- 设置界面通用部分重构：开关按钮移至语言/引擎选择上方，灰色细线分隔
- 开关打开颜色从硬编码绿色改为跟随主题色（`theme.themeColor`）
- 设置窗口不再始终置顶
- LLM 引擎提示词优化：语言代码映射为自然语言名称，同时指定源语言和目标语言

### 修复
- 修复翻译窗口持续显示"翻译中..."卡死的问题（Rust 引擎添加 10s/30s 超时，前端添加 30s 超时 + AbortController + isLoadingRef 竞态修复）
- 修复设置反序列化失败导致设置无法保存的问题（`image_translation_enabled` 添加 `#[serde(default)]`）
- 修复大模型API/Ollama 翻译时 LaTeX 公式无法正常显示的问题

## [0.0.4] - 2026-04-15

### 修复
- 修复大模型API/Ollama 翻译时 LaTeX 公式无法正常显示的问题
- LLM 引擎改为直接发送原始文本，不再使用占位符提取（LLM 通过提示词原样保留 LaTeX）
- 非 LLM 引擎（百度、谷歌）保持原有占位符提取机制

### 功能增强
- **LLM 提示词优化**：语言代码映射为自然语言名称（`zh` → `Chinese`），提示词与设置语言同步
- **源语言支持**：LLM 提示词现在同时指定源语言和目标语言（自动检测时仅指定目标语言）
- 新增 `lang_code_to_name()` 公共函数，供 llmapi/ollama 引擎共用
- 提示词末尾添加"仅输出翻译结果"约束，减少模型附带说明文字

### 重构
- 将硅基流动引擎（siliconflow）重命名为大模型API（llmapi）
- 前端显示名称更新为"大模型API"，设置键名更新为 `llmapi_api_key` / `llmapi_model`
- 移除百度翻译 LLM API 支持，仅保留标准通用翻译API（MD5 签名认证）

## [0.0.3] - 2026-03-31

### 修复
- 修复翻译结果显示 `Parse("No translation result")` 错误
- 修复标题栏引擎下拉框无法点击选择的问题
- 修复引擎选择在标题栏和设置窗口之间不同步的问题
- 修复外观设置修改后翻译窗口不生效的问题

### 功能增强
- **Google 翻译双 URL 支持**：镜像源 URL + 官方 API URL + API Key，自动降级
- **设置自动保存**：2 秒防抖自动保存，关键设置（引擎/语言）立即保存
- **跨窗口设置同步**：修改引擎或语言后，翻译窗口实时响应
- **错误提示优化**：引擎错误信息改为中文友好提示，凭证缺失时提前校验

### 技术改进
- 后端配置向后兼容：旧版 `google_url` 自动迁移到新字段
- 前端事件监听：`theme-changed` 事件实现跨窗口状态同步
- Radix UI 下拉框修复：`onClick` 改为 `onSelect` + `e.preventDefault()`
